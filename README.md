# 四则运算练习程序 - Alpha20251004 (第三代)

一个功能丰富的数学练习软件，支持基础四则运算、多项式练习和九九乘法表，可自定义题目难度和数量。

## 功能特点

### 🎯 核心功能
- **基础运算**：加法、减法、乘法、除法
- **难度分级**：个位、十位、百位、千位、万位数
- **题目定制**：自由选择题目数量
- **成绩记录**：自动保存练习成绩到文件
- **九九乘法表**：完整的乘法口诀表显示

### 🆕 新增特性
- **多项式支持**：支持多位数连续运算（开发中）
- **混合运算**：加减乘除组合练习（规划中）
- **题目导出**：生成本地练习题文件（基础框架已实现）

### 📊 智能特性
- **实时反馈**：每题立即判断对错
- **鼓励机制**：随机显示鼓励或表扬语句
- **进度显示**：显示剩余题目数量
- **历史回顾**：查看以往练习成绩
- **精度控制**：除法结果保留两位小数

### 🛠 技术特色
- **跨平台兼容**：支持 Windows 和 Linux 系统
- **输入验证**：完善的错误处理和输入检查
- **用户友好**：清晰的操作指引和界面
- **多语言支持**：完整的中文界面和提示

## 使用方法

### 启动程序
```bash
# Windows
Alpha20251004a3.exe

# 或重命名为
四则运算.exe
```

### 操作指南

1. **选择运算类型**：
   - 加法: 1 | 减法: 2 | 乘法: 3 | 除法: 4
   - 九九乘法表: 5 | MIT许可证: 6
2. **配置题目参数**：
   - 输入两个位数参数和题目数量
   - 个位: 1, 十位: 2, 百位: 3, 千位: 4, 万位: 5
3. **解答题目**：依次回答生成的数学题
4. **查看总结**：获得成绩反馈和详细统计

### 示例



```text
输入：1 2 5
生成：5道加法题，前项为个位数，后项为十位数

输入：3 1 4  
生成：4道除法题，前项为百位数，后项为个位数
```
